**Crabbyqlite** is a SQLite/SQLCipher database browsing tool that allows you to browse and query SQLite and SQLCipher database files with a GUI. I decided to build **Crabbyqlite** after I found out that the
SQLite database browsing tool [I was using](https://sqlitebrowser.org/) required manually building the project from source and wasn't apart of the regular releases (It's a massive C++/C repository so you're not going to catch me trying to build that). Instead I decided to create my own one in **Rust** using **GPUI** which makes it easy to build!
